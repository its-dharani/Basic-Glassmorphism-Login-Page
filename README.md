# Basic Glassmorphism Login Page

## Description
A login form UI built using HTML and CSS, inspired by glassmorphism design.  
The project focuses on clean layout, modern styling, and hover effects without using JavaScript.

## Features
- Glassmorphism effect using backdrop-filter
- Centered login container using Flexbox
- Background image with gradient overlay
- Custom icons using Boxicons
